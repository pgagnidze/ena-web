# Ena Website

Website for Ena programming language. Features:

- AI helper.
- Web interpreter

## Running Locally

Here's a quick overview of how to run it locally.

- Clone repo

```bash
git clone https://github.com/pgagnidze/ena-web.git
```

- Install dependencies

```bash
npm i
```

- Run app

```bash
npm run dev
```
